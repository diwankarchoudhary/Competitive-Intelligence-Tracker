# AI Usage Notes

AI was used as a coding assistant during development for:
- Architecture planning
- FastAPI route structuring
- Diff summarization logic
- Frontend data flow suggestions

All AI-generated code was reviewed, tested, and modified manually.

## LLM Used
- Provider: OpenAI
- Model: GPT-4-class (via OpenAI API)

## Why this model
- Strong summarization ability
- Good handling of structured diffs
- Reliable API and tooling support

## What was manually verified
- API routes and responses
- Diff correctness
- Error handling
- Frontend-backend integration
- CORS and deployment readiness
